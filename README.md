# Análise de Dados: Educação e Renda no Nordeste

> Um estudo estatístico sobre a correlação entre a taxa de analfabetismo e a renda média domiciliar nos estados do Nordeste brasileiro.

##  [Clique aqui para ver o Relatório Interativo (Site)](https://rqueiroga.github.io/projeto-nordeste/projeto_analise_nordeste.html)

---

### O Problema
O Nordeste enfrenta desafios históricos de desigualdade. O objetivo deste projeto foi responder com dados: **"A educação básica impacta diretamente no bolso do cidadão?"**

### Tecnologias Utilizadas
* **Python 3.12**
* **Pandas** (Tratamento e organização dos dados)
* **Matplotlib** (Visualização de dados)
* **HTML/CSS** (Automação de relatórios web)

### Principais Resultados
Os dados da PNAD Contínua (IBGE 2023) revelaram uma forte correlação negativa:
* **Rio Grande do Norte:** Menor taxa de analfabetismo (9.5%) ➔ Maior Renda (R$ 1.625).
* **Maranhão/Alagoas:** Maiores taxas de analfabetismo (>12%) ➔ Menores Rendas.
